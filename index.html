How does JavaScript Work?

JavaScript is a client-side programming language that means it's run inside a browser. The browser has a built-in interpreter that reads our JavaScript code if found on the web page and runs it. The interpreter starts when the browser loads the pages and it reads line-by-line instructions and executes them. Every browser has the engine to run JavaScript Code. Such as - Google has a V8 engine. The V8 engine is used on Chrome and Node js. The Engine consists of two main components - i) Memory Heap ii) Call Stack. Memory Heap is the place where the memory allocation happens and Call Stack is the place where our stack frames are as our code executes. JavaScript is a single-threaded programming language means is has single call stack. Therefore javascript can do only one thing at a time.
